# ToDo List

## 🛠️ Tecnologias

- [React](https://reactjs.org/)
- [C#](https://learn.microsoft.com/dotnet/csharp/)
- [ASP.NET Core](https://learn.microsoft.com/aspnet/core/)
- [Visual Studio Code](https://code.visualstudio.com/) / Visual Studio
